# Previsão de Evasão e Sucesso Acadêmico de Estudantes

## 📌 Descrição e Objetivo
Este repositório contém o projeto final de Aprendizado de Máquina focado na previsão do status acadêmico de estudantes (`Dropout`, `Enrolled`, `Graduate`) a partir de dados demográficos, socioeconômicos e de desempenho nos primeiros semestres.

* **Fonte dos Dados:** [UCI Machine Learning Repository - Dataset #697](https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success)
* **Tipo da Tarefa:** Classificação Multiclasse.
* **Modelos Utilizados:** `SGDClassifier` (Baseline) e `RandomForestClassifier` (Modelo Escolhido).
